# All-Prime
isPrimeXmetoduna  gönderilen  argumanın  asal  olup  olmadığı  test  edilecek, eğer  sayı  asal  ise  bu  kez  sayının  basamak 
değerleri  toplanarak  elde edilen  sayının asal  olup  olmadığı  test  edilecektir. Bu  işlem  sonuçta  tek  basamaklı  bir  sayı
kalana  kadar  devam  edilecektir. Eğer  en  son  elde  edilen  tek  basamaklı  sayı  dahil  tüm  sayılar  asal ise isPrimeXmetodu 
true  değerine  geri  dönecektir. Eğer  herhangi  bir  kademede  asal  olmayan  bir  sayı elde  edilirse  metot  false değerine
geri  dönecektir.
